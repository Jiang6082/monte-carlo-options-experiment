# Changelog

## 0.2.0 - Portfolio Upgrade

- Renamed the package to `mc_options`.
- Added GitHub Actions CI, Ruff, mypy, and pre-commit configuration.
- Added CLI commands for pricing and implied volatility.
- Added moment matching, Sobol draws, Brownian bridge paths, and control variates.
- Added digital and barrier option support.
- Added discrete geometric Asian benchmark and implied-volatility solver.
- Added examples, docs site skeleton, saved figures, and performance benchmark script.
- Added analytical continuous barrier benchmark, Longstaff-Schwartz American pricing, Heston simulation, synthetic implied-volatility smile calibration, coverage reporting, notebook execution in CI, mkdocstrings API docs, and benchmark artifacts.
- Added a static interactive dashboard for visual exploration of validation, convergence, variance reduction, Asian options, Greeks, advanced models, and browser-side Black-Scholes versus Monte Carlo pricing.

## 0.1.0 - Initial Project

- Implemented vectorized GBM simulation.
- Added European and arithmetic Asian option payoffs.
- Added Black-Scholes benchmarks, Monte Carlo confidence intervals, Greeks, notebooks, and tests.
